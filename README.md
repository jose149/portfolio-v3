# Portfolio v3

Initial pnpm and Turborepo workspace for the portfolio platform.

## Requirements

- Node.js `>=24.13.1`
- pnpm `10.30.0`

## Setup

Install workspace dependencies:

```bash
pnpm install
```

Start the web application in development mode:

```bash
pnpm dev
```

Run TypeScript checks across the workspace:

```bash
pnpm typecheck
```

Build the web application:

```bash
pnpm build
```
